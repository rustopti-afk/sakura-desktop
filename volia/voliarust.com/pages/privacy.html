Конфіденційність

Як ми збираємо, використовуємо та захищаємо ваші персональні дані

Остання редакція: 22 квітня 2026 року.
Ми серйозно ставимось до захисту вашої приватності. Ця сторінка пояснює, які дані ми збираємо і навіщо.

1. Які дані ми збираємо

Тип даних Що саме Навіщо
Steam профіль SteamID64, нікнейм, аватар (публічні дані Steam) Ідентифікація гравця на сервері та сайті
Ігрова активність Час гри, статистика, дії на сервері Адміністрування, античіт, підтримка
Покупки Нікнейм, перелік куплених товарів, дата Активація привілеїв, підтримка клієнтів
Технічні дані IP-адреса, апаратний ID (HWID) Захист від мульти-акаунтів та банів
Дані сайту localStorage браузера (налаштування, сесія) Збереження мови, теми та стану авторизації

Ми не збираємо реальне ім'я, email, телефон або платіжну інформацію. Платежі обробляються через захищені сторонні сервіси (LiqPay, Fondy тощо).

2. Передача даних третім особам

Ми не продаємо і не передаємо ваші персональні дані стороннім компаніям для маркетингу чи реклами.

Ваші дані можуть бути передані лише у виняткових випадках:

  • На законну вимогу правоохоронних органів України
  • Платіжним системам — мінімально необхідний обсяг для обробки транзакцій
  • Хостинг-провайдерам — для технічного функціонування серверів

3. Cookies та localStorage

Наш сайт використовує localStorage браузера (аналог cookies) для:

  • Збереження обраної мови інтерфейсу (uk / en)
  • Зберігання стану авторизації (Steam нікнейм та аватар)
  • Налаштувань сайту, обраних адміністратором

Ці дані зберігаються виключно у вашому браузері і не передаються на наші сервери. Ви можете очистити їх у будь-який час через налаштування браузера.

Ми також використовуємо сторонні сервіси, які можуть встановлювати власні cookies:

  • BattleMetrics API — для відображення статистики сервера (без ідентифікації користувача)
  • Steam OpenID — для авторизації через Steam
  • Google Fonts — для завантаження шрифтів

4. Безпека даних

Ми вживаємо розумних технічних та організаційних заходів для захисту ваших даних:

  • HTTPS-шифрування для всіх з'єднань із сайтом
  • Обмежений доступ до серверних баз даних (тільки адміністрація)
  • Регулярне резервне копіювання даних серверу

Зверніть увагу: жоден метод передачі даних через інтернет не є на 100% безпечним. Ми не можемо гарантувати абсолютну безпеку.

5. Ваші права

Відповідно до законодавства України, ви маєте право:

  • Знати — які ваші дані ми зберігаємо
  • Виправити — неточні дані (нікнейм тощо)
  • Видалити — ваш профіль та ігрові дані (зверніться в підтримку)
  • Заперечити — проти обробки даних, що не є необхідною для роботи сервісу

Для реалізації будь-якого з цих прав зверніться до нас через Discord або форму підтримки.

6. Діти

Наші сервіси не призначені для осіб молодше 13 років. Якщо вам стало відомо, що дитина молодше 13 років надала нам персональні дані — будь ласка, зв'яжіться з нами для їх видалення.
